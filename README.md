# Paulenzki Website

This is the repository for the Paulenzki website.
Paulenzki is an Acrobatics Duo based in Regensburg, Germany.
Paulenzki is a combination of the names Paula and Lenz

## Structure of the Website

The website is structured in a way that allows for easy navigation and content management. The main sections include:

- **Home**: The landing page with an introduction to Paulenzki.
- **About**: Information about the duo, their background, and their journey.
- **Gallery**: A collection of photos and videos showcasing their performances.
- **Contact**: A form for inquiries and bookings.

It also contains an impressum and a privacy policy to comply with legal requirements.

Navigation is facilitated through a top navigation bar that remains consistent across all pages.
The footer includes links to social media profiles and legal information.

The Website is designed to be responsive, ensuring a good user experience on both desktop and mobile devices.
All styling is done using Tailwind CSS, which allows for a clean and modern design.

The Webpage is in German and English.
